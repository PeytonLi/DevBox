## Managed Agent Notes

- The backend defaults to `DEVBOX_GENAI_MODE=auto`, using a live Managed Agent only when `GEMINI_API_KEY` is present.
- The default managed agent id is `antigravity-preview-05-2026`.
- The default remote tool override is `code_execution,url_context`.
- `google_search` is not enabled by default. It can be explicitly added in the UI or API request if the demo requires it.
- Local development and tests use a deterministic simulator when preview access or credentials are unavailable.
